College Club Web Application

Problem Statement

College clubs often lack a professional and engaging online presence to attract and manage members effectively. The absence of a user-friendly interface results in difficulties for users to access event details, join activities, or communicate with the club.

Solution

Designed an intuitive and visually appealing user interface for the club's web application using Figma. The design prioritizes ease of navigation, clear communication of event details, and a modern aesthetic to engage users effectively.

Project Images

Tools Used

  • Figma: For designing the user interface.
  • HTML/CSS: For translating the design into a functional web application.
